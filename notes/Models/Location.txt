====== Location ======
Created Wednesday 28 May 2014

What SB calls a Room, but it covers more than rooms.

Fields

id			integer	Rails's id for the location
short_name	string	Short name for the location (e.g. CMR)
name		string	Longer name (e.g. Charles Maude Room)
active		boolean	Flag indicating whether the location is active
